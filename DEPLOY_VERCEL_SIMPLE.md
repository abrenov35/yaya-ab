# ⚡ Vercel En 1 Clic

Déployer votre index YAYA en **moins de 2 minutes**!

---

## 🎯 Les 3 Étapes

### **1️⃣ Aller sur Vercel**

Cliquez ce lien: https://vercel.com/new

(Ou allez sur https://vercel.com et cliquez "Add New Project")

### **2️⃣ Sélectionner le Repo GitHub**

1. Vercel vous demande: "Quel repo?"
2. Cherchez: **`yaya-ab`**
3. Cliquez dessus

### **3️⃣ Déployer**

1. Vercel configure tout automatiquement
2. Cliquez **"Deploy"**
3. ⏳ Attendre 1-2 minutes
4. ✅ **C'est en ligne!**

```
Visit: https://yaya-ab.vercel.app
```

## 🔄 Auto-Redéploiement

Chaque fois que vous modifiez le code sur GitHub:
```bash
git push origin main
```

Vercel redéploie **automatiquement** en 1-2 minutes! ✨

## 📊 Google Sheets + YAYA

Une fois déployé sur Vercel:

1. Allez sur Google Sheets: https://sheets.google.com
2. Créez une feuille "YAYA"
3. Outils → Éditeur de script
4. Collez le code de `google-apps-script.gs`
5. Changez l'URL en:
   ```javascript
   const API_URL = "https://yaya-ab.vercel.app";
   ```
6. Enregistrez et testez le menu "📊 YAYA"

**Voir le guide complet:** `VERCEL_APPSCRIPT.md`
