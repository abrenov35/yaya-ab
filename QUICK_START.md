# ⚡ Démarrage Rapide - 3 Minutes

## Pour les Pressés

### 1️⃣ Ouvrir Terminal ici
Clic droit sur le dossier → "Ouvrir Terminal Windows ici" (ou "Terminal" sur Mac/Linux)

### 2️⃣ Installer & Démarrer
```bash
npm install && npm start
```

### 3️⃣ Ouvrir le Navigateur
Allez à: `http://localhost:3000`

✅ **C'est prêt !**

---

## Vous Avez une Erreur ?

### "npm not found"
→ Installez Node.js: https://nodejs.org/

### "Port 3000 already in use"
```bash
npm start -- --port 8080
```

### "Missing files"
```bash
npm install
```
